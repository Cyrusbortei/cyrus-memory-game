# Memory Card Flip Game A brain training memory matching game with 3 difficulty levels and star rating system.  ## Features - 3 Difficulty Levels: Easy 4x4, Medium 4x4, Hard 5x4 - 3D Card Flip Animations with match pulse effect - Move counter + Timer + Star rating system - Best score saved locally on device - 100% offline - works without internet.
